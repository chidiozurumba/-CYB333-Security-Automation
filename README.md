# Password Strength Analyzer

## Project Overview
The Password Strength Analyzer is a Python script that was created for CYB 333 - Security Automation at National University. 
The program will analyze the strength of a password and provide information on how strong it is.

This project will show some basic secuirty automation concepts and help users understand how to create a basic automation script.

---

## Features
- Scores passwords from 0 to 100
- Checks minimium password length
- Detects upderscase and lowercase letters
- Checks for numbers and special characters
- Checks for commonly used passwords
- Gives suggestions on how to improve passwords

---

## How the Script Works
1. User inputs a password.
2. The script will then analyze the password for weaknesses.
3. A stregnth score is displayed to the user.
4. Suggestiions are then provided if the password is weak.

## Requirements
- Python 3.x
- No external libraries are needed

The program only uses built-in modules.

---

## How to Run the Program
1. Download this repository from GitHub.
2. Open a terminal or command prompt.
3. Navigate to the project directory.
4. Run the program using python password_strength_analyzer.py
5. Enter a password when prompted.
6. View the password strength score and get feedback on the password.
